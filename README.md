# 🚀 AI Content Script Generator

An automated script designed for content creators, marketers, and social media managers. By combining **multimodal image briefs** and **text prompts**, this AI-powered tool generates a complete content blueprint—including hooks, scene-by-scene storyboards, creative ideas, and optimized hashtags.

---

## ✨ Features

* 📸 **Multimodal Input:** Drop an image brief (moodboard, reference, sketch) along with your text prompt.
* 🎬 **Scene-by-Scene Breakdown:** Generates a structured narrative, complete with visual directions and voiceovers (VO).
* 💡 **Idea Multiplier:** Gives alternative angles and hooks to maximize retention.
* 🏷️ **Smart Hashtags:** Auto-generates trending and relevant hashtags tailored to your target audience.
